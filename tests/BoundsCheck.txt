Bounding Box Checking
=====================

See http://trac.gispython.org/projects/PCL/ticket/127.

Adding with bogus bounds
------------------------

  >>> import rtree
  >>> index = rtree.Rtree()
  >>> index.add(1, (0.0, 0.0, -1.0, 1.0))
  Traceback (most recent call last):
  ...
  RTreeError: LASError in "Index_InsertData": IllegalArgumentException: Region::initialize: Low point has larger coordinates than High point.

  >>> index.intersection((0.0, 0.0, -1.0, 1.0))
  Traceback (most recent call last):
  ...
  RTreeError: LASError in "Index_Intersects": IllegalArgumentException: Region::initialize: Low point has larger coordinates than High point.

Adding with invalid bounds argument should raise an exception

  >>> index.add(1, 1)
  Traceback (most recent call last):
  ...
  TypeError: Bounds must be a sequence
